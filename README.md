\# Maze Solver Game Using Artificial Intelligence



This project is an AI-based maze solver game implemented in Python. It demonstrates how search algorithms can be used to find the shortest path from a start point to a goal in a randomly generated maze.



\## Features



\- Random maze generation

\- Manual player movement

\- Breadth-First Search (BFS) algorithm

\- A\* search algorithm

\- Comparison between BFS and A\*

\- Interactive execution using Google Colab



\## Technologies Used



\- Python

\- Google Colab

\- BFS Algorithm

\- A\* Algorithm

\- Manhattan Distance heuristic

\- ipywidgets



\## How It Works



1\. A random maze is generated.

2\. The start and goal positions are placed.

3\. The user can move manually or let the AI solve the maze.

4\. BFS and A\* search for the shortest path.

5\. The algorithms are compared based on efficiency.



\## Algorithms



\### Breadth-First Search

BFS explores nodes level by level and guarantees the shortest path.



\### A\* Search

A\* uses the Manhattan Distance heuristic to reach the goal more efficiently while still finding an optimal path.



\## How to Run



Open the notebook in Google Colab and run the cells from top to bottom.
